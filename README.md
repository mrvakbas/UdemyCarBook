# 🚗 CarBook - Full Stack Araç Kiralama Platformu
CarBook, modern web teknolojileri ve kurumsal yazılım mimarileri kullanılarak geliştirilmiş, uçtan uca bir araç kiralama çözümüdür. Proje, hem kullanıcılar için akıcı bir kiralama deneyimi hem de yöneticiler için veriye dayalı gelişmiş bir yönetim paneli sunar.
________________________________________
## 🛠️ Teknolojik Altyapı
Proje, sürdürülebilir, test edilebilir ve yüksek performanslı bir yapı oluşturmak adına aşağıdaki teknolojilerle inşa edilmiştir

| Katman       | Teknolojiler                                                                 |
|--------------|------------------------------------------------------------------------------|
| Backend      | ASP.NET Core MVC (.NET 8.0), C#, Entity Framework Core, LINQ                |
| Mimari       | Onion Architecture, CQRS, MediatR                    |
| Güvenlik     | JWT, Rol bazlı erişim                    |
| Veritabanı   | Microsoft SQL Server – ilişkisel veri yapısı ve performans odaklı sorgular  |
| Frontend     | HTML5, CSS3, Bootstrap 5 – duyarlı ve modern tasarım bileşenleri            |
| UI/UX        | jQuery, Font Awesome – etkileşimli bildirimler ve ikon desteği |
________________________________________

## ✨ Öne Çıkan Özellikler
## 👤 Kullanıcı Deneyimi
•	Hızlı Rezervasyon: Lokasyon ve tarih bazlı anlık araç arama motoru.
•	Şeffaf Fiyatlandırma: Günlük, haftalık ve aylık bazda detaylı fiyat paketleri.
•	İçerik Dünyası: Otomotiv dünyasına dair güncel blog yazıları ve kullanıcı yorumları.
•	Güven Odaklı: Gerçek kullanıcı referansları ve platform istatistikleri.
## 🛡️ Yönetim Paneli (Admin)
 •	Veri Analitiği: Araç sayıları, lokasyon verileri ve kazanç grafiklerinin merkezi takibi. 
 •	Tam Kontrol: Araç, marka, lokasyon ve kategori bazlı kapsamlı CRUD işlemleri. 
 •	İçerik Yönetimi: Blog içeriklerinin, yazar profillerinin ve yorumların moderasyonu. 
 •	Sistem Ayarları: Site genelindeki iletişim ve sosyal medya bilgilerinin dinamik yönetimi.
